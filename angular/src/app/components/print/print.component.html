Storage
Store all items in a cool, dry, protected place with easy access in an emergency.
  • Store the supplies near an exit to facilitate loading into vehicles if forced to leave quickly.

  • Storage place depends on the risks prone to the area:
    Wind storms : keep the supplies in the basement
    Flooding : store items on upper floors
    Smaller homes and apartments : concern about the store space
